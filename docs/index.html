UINotifications

UINotifications makes it really easy to create in-app UINotifications.

Example

To run the example project, clone the repo, and run pod install from the Example directory first.

Requirements

  • Swift 3.0+
  • iOS 8.0+

Installation

UINotifications is available through CocoaPods. To install it, simply add the following line to your Podfile:

pod "UINotifications"
